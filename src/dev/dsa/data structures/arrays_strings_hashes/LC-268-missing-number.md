# 268. [Missing Number]()
# O(n)



-  js (naive) 

```js 

var missingNumber = function (nums) {
    let m = null;

    for (let i = 0; i <= nums.length; i++) {
        let check = i;
        if (nums.indexOf(check) == -1) {
            m = i;
        }
    }

    return m;


};

```

/**
